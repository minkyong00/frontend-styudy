제어문(Control Statement)
- 프로그램의 동작 흐름에 조건/ 반복을 통해 제어할 수 있는 실행문
- 표현식의 결과에 따라 별도릐 명령을 수행하도록 제어하는 문장

▶조건문
	▷if문
	if(조건식){
		실행할 문장
	}
	위의 조건식 결과에 상관없이 무조건 검사한다.
	if(조건식){
		실행할 문장
	}
	if(조건식){
		실행할 문장
	}


	if(조건식){
		실행할 문장
	}else if(조건식){
		위 조건식이 false일 경우만 검사, 아니면 검사하지 않는다.
		실행할 문장;
	}
	...
	}else{
		실행할 문장
	}



	▷switch문	
	switch(값){
	case 값1:
		값1 일때 실행할 문장
		break;
	}
	case 값2:
		값2 일때 실행할 문장
		break;
	}
	...
	default:
		실행할 문장;
		break;
	}

기타 제어문
	break : 즉시 해당 중괄호 영역을 탈출한다.
	- if문 안에서 사용 시 if문을 탈출하지 않고 if문을 감싸고 있는 중괄호 영역을 탈출한다.

	continue : 즉시 다음 반복.
	- 아래에 작성된 코드를 실행하지 않고 다음 반복을 실행한다.
